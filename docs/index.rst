========
pipeline
========

pipeline is a library containing common utilities used by the components of the
Ingestion Pipeline.

Installation
============

If you are using the internal package index server, you can install pipeline
using Pip::

    $ pip install pipeline

Otherwise, pipeline can be installed from source::

    $ python setup.py install
